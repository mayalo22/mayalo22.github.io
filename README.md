# אתר הספרים של מעיין גלעד

דף נחיתה סטטי, מותאם לנייד, עם מחירים לפי פורמט וחנות, רכישה ישירה, קישורי קהילה וקישור למשחק של „כוכבים רואים רק בחושך”.

## פרסום בענן ועדכון אוטומטי

1. יוצרים מאגר GitHub חדש ומעלים אליו את כל תוכן התיקייה הזאת (כך ש־`index.html` נמצא בשורש המאגר).
2. במאגר נכנסים אל **Settings → Pages** ובוחרים **Source: GitHub Actions**.
3. נכנסים אל **Actions**, פותחים את הפעולה **Update book prices** ומריצים אותה פעם אחת ידנית.

מכאן האתר מתארח ב־GitHub Pages והפעולה בודקת מדי יום אם חלפו 72 שעות. כאשר חלפו שלושה ימים היא מרעננת את המחירים, שומרת את הנתונים ומפרסמת את האתר מחדש. המחשב האישי לא צריך להיות פתוח.

## הערות חשובות

- המחיר הקובע תמיד הוא המחיר שמופיע באתר החנות בזמן הרכישה.
- אם חנות חוסמת זמנית גישה אוטומטית או משנה את מבנה האתר, נשמר המחיר האחרון שנבדק במקום למחוק אותו.
- קישור לקבוצת הוואטסאפ מופיע בקובץ `index.html`. אם משתנה קישור ההזמנה, יש לעדכן אותו שם.
- הרכישה הישירה של „ילדה שקופה” מציגה תשלום של 50 ₪ באמצעות קוד QR או קישור ישיר ל־Bit ול־PayBox, או למספר 050-241-0393, ולאחריו שליחת אישור וכתובת למשלוח בוואטסאפ.

## בדיקה מקומית

יש להפעיל שרת קבצים מקומי מתוך התיקייה (למשל `python -m http.server 8000`) ולפתוח `http://localhost:8000`. פתיחה ישירה של `index.html` עלולה למנוע מהדפדפן לקרוא את `data.json`.
